
# Особенности реализации
## Иерархия геометрии
- Реализованы основные кривые, согласно ТЗ.
- Добавлены типы `Point`, `Line` и `LineSegment`. В них вырождаются кривые при передаче некорректных параметров.

## Настройка точности 
- Вспомогательная структура `GeometryConfig` позволяет задать эпсилон при сравнении чисел с плавающей точкой. 
- Конфигурация передается в `GeometryBuilder` и конструкторы кривых.

## Парсинг файлов
- Чтение и парсинг файлов обеспечивается функцией `GeometryLoader::LoadFromFile()`.

## Создание объектов
- Для создания геометрии используется класс `GeometryBuilder`.
- Его задача - исправление некорректрых параметров перед вызовом конструкторов кривых.
- При необходимости кривые заменяются на фигуры другого типа.

# Сборка
## Требования
Сборка проверялась на GCC (Windows, Linux) и MSVC.

Сборка требует:
- GCC 14 или выше
- MSVC 19.37 или выше
## Процесс сборки
Сборка через CMake
```
cmake -S . -B build
cmake --build build
```
